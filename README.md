FitBuddy – AI Powered Fitness Planner

FitBuddy is an AI-powered web application that generates personalized workout plans and nutrition tips based on a user’s fitness goals. 
The platform helps users achieve objectives such as weight loss, muscle gain, or general wellness by providing structured and customized 7-day workout routines.

The application is built using FastAPI, Google Gemini AI, and SQLite. Users enter their personal details such as age, weight, fitness goal, and preferred workout
intensity to receive a customized workout schedule generated by AI. The platform also supports user feedback to update workout plans dynamically.

Features:-

AI-generated personalized 7-day workout plans
Goal-based nutrition and recovery tips
Feedback system for improving workout plans
FastAPI backend with REST API endpoints
Simple HTML frontend interface
SQLite database for storing user profiles and plans

Technologies Used:-

Python
FastAPI
Google Gemini AI
SQLite
HTML / CSS
Jinja2 Templates
